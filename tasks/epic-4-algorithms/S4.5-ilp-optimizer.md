# S4.5 — ILP optimizer (OR-Tools)

- **Epic:** 4 — Algorithms
- **Story points:** 5
- **Status:** ☑ Done
- **Depends on:** S3.1, S3.2, S4.2

## Goal
An exact/near-exact constraint optimizer using OR-Tools CP-SAT, to compare against
local search.

## Tasks
- [ ] Decision vars `x[p,g] ∈ {0,1}` (participant p in group g)
- [ ] Hard constraints: Σ_g x[p,g] = 1; Σ_p x[p,g] = group_size
- [ ] Linearize soft objectives where possible (skill balance via deviation vars,
      preference rewards, diversity spread)
- [ ] Objective mirrors the weighted cost (S4.2) as closely as linearization allows
- [ ] Time limit + return best feasible solution
- [ ] Map solver result back to an `Assignment`

## Acceptance criteria
- Produces feasible assignments validated by S3.1.
- On small instances, matches or beats local search on cost.
- Respects the configured time limit and reports solver status.
