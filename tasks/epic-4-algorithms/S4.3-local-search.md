# S4.3 — Local search optimizer (swap-based hill climbing)

- **Epic:** 4 — Algorithms
- **Story points:** 5
- **Status:** ☑ Done
- **Depends on:** S4.1, S4.2

## Goal
Iteratively swap participants between groups, accepting swaps that lower cost.

## Tasks
- [ ] Start from a feasible random assignment
- [ ] Propose a swap of two participants in different groups
- [ ] Accept if delta cost < 0 (greedy hill climbing)
- [ ] Stop on max iterations or no-improvement plateau
- [ ] Record cost trajectory for convergence plots (S5.3)

## Acceptance criteria
- Never produces an infeasible assignment (swaps preserve group sizes).
- Final cost ≤ initial cost.
- Reproducible with a seed; trajectory exposed for plotting.
