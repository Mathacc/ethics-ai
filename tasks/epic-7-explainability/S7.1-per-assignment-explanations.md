# S7.1 — Per-assignment explanations

- **Epic:** 7 — Explainability
- **Story points:** 5
- **Status:** ☑ Done
- **Depends on:** S3.2, S4.2

## Goal
Explain *why* each participant landed in their group, in terms of the soft constraints.

## Tasks
- [ ] For each participant: marginal cost contribution of their placement
- [ ] Counterfactual: best alternative group and the cost delta of moving there
- [ ] Per-group summary: which soft constraints it satisfies well/poorly
- [ ] Structured explanation object (factors + weights + scores)

## Acceptance criteria
- Explanations derived from the actual cost terms (faithful, not post-hoc invented).
- Counterfactual move is feasible and correctly scored.
- Covered by unit tests on small instances.
