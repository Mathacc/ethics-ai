# S3.2 — Soft constraints as objectives

- **Epic:** 3 — Constraint modeling
- **Story points:** 3
- **Status:** ☑ Done
- **Depends on:** S3.1

## Goal
Express soft goals as scorable penalty/utility terms (lower = better, or normalized).

## Tasks
- [ ] Skill balance: minimize variance of group skill means across groups
- [ ] Diversity: reward even spread of diversity attributes per group
- [ ] Preference satisfaction: reward honoring likes / penalize dislikes co-grouping
- [ ] Each term normalized to a comparable scale [0,1]
- [ ] Pure functions: `term(assignment) -> float`

## Acceptance criteria
- Each term is independently testable with hand-built assignments.
- Terms normalized so weights in the cost function are meaningful.
- Documented direction (penalty vs reward) for each term.
