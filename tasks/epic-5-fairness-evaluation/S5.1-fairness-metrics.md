# S5.1 — Fairness metrics

- **Epic:** 5 — Fairness evaluation
- **Story points:** 5
- **Status:** ☑ Done
- **Depends on:** S1.2

## Goal
Quantitative, optimizer-agnostic metrics to judge an assignment's fairness.

## Tasks
- [ ] Skill-balance: variance of per-group skill means across groups (per skill + aggregate)
- [ ] Experience balance: spread of experience levels across groups
- [ ] Diversity spread: how evenly diversity attributes distribute per group
- [ ] Preference satisfaction rate: % of honored preferences
- [ ] `evaluate(assignment) -> dict[str, float]`

## Acceptance criteria
- Metrics computed independently of the cost function (avoid circular evaluation).
- Hand-built best/worst cases produce expected extreme values.
- Documented formula for each metric (feeds the report).
